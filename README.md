Lunchbox Sustenance Project

Welcome to the Lunchbox Sustenance Project! This mod brings an immersive cooking experience to the Minecraft world. Tired of the same old roasted meat and carrots? It's time to put on your apron, light the fire, and become a true blocky world chef!

This mod is inspired by the game Don't Starve and brings the cooking pot to Minecraft.

At the heart of the Lunchbox Sustenance Project is a cooking pot! It allows you to explore and create over 40 unique and delicious dishes by combining a variety of different ingredients.

1.COOKING POT!
·It has four input slots, allowing for free combination of ingredients.
·It uses a unique attribute value system to determine recipes, rather than combinations of specific items, so the theoretical number of recipes is very large.

2.Food attribute value system
·We've assigned attribute tags to almost all available ingredients, such as:

·Meat Value - Beef, Pork, Chicken, etc.

·Vegetable Value - Carrots, Potatoes, Beetroot, etc.

·Fruit Value - Apples, Watermelons, Berries, etc.

·Fish Value - Salmon, Snowfish, etc.

·Egg Value - Eggs, etc.

3.Explore and Discover
·There are no fixed crafting recipes! Recipes depend on the attribute values ​​achieved by the combination of ingredients you put into the cooking pot. For example, meat and eggs can cook bacon and eggs, while meat and honey can make honey ham.

·There are theoretically thousands of recipe combinations, encouraging players to experiment and discover hidden recipes!

·Each dish has a unique name and a carefully designed food recovery effect.

·If your recipe cannot make any dish, it will produce a "failed dish." Don't be discouraged, try again next time!

·Recipes have different priorities. If a recipe meets the requirements of multiple dishes, the higher-priority dish will be made first.

·Probability theory applies. If multiple dishes support the same recipe and have the same priority, one will be randomly made (e.g., bacon and eggs versus lasagna, a nod to the game *Don't Starve*).

4.JEI ADDON
Supports querying recipes using JEI modules. While the current viewing options may not be very flexible, we've done our best.

5.Data Pack Support!
·We understand the importance of customization, so this mod fully supports extending and modifying all content via data packs!

·Custom Food Attributes: You can easily define cooking attributes (such as meat value, vegetable value, etc.) for any vanilla or other mod item. Want Nether Stars as seasoning? No problem!

·Add New Recipes: Create entirely new dishes using only JSON files, without modifying the code. You can precisely specify the various combinations of attribute values ​​you need and define their output.

·Modify Existing Content: Think a recipe is too powerful or unreasonable? You can directly override and adjust it via data packs.

By default, ingredients from the Farmers' Delight mod are supported.
